## Ryan Chu's CodePath Prework

### App Description
Ryan Chu's Prework project for CodePath's Spring 2021 iOS Mobile App Development course. This project was created using XCode and has 4 main functions: editing a label & displaying it, changing view color, changing text color, and refreshing the app to default text/background color.

- [x] 1. User sees custom text in a label - Hello from {name}!
- [x] 2. User see's custom background color.
